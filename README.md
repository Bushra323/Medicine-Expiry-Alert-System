**Medicine Expiry Alert System**

## Project Description

The Medicine Expiry Alert System is a desktop-based application developed using Java, Java Swing, JDBC, and MySQL following the MVC (Model-View-Controller) architecture. The system helps users check medicine expiry information and enables pharmacists to efficiently manage medicine inventory.

The application provides separate interfaces for users and pharmacists. Users can search medicines using medicine details and receive expiry alerts. Pharmacists can manage inventory by adding, updating, deleting, and viewing medicine records while monitoring expired and near-expiry medicines.

* User Registration

### Pharmacist Module

* View Medicines
* Add Medicines
* Update Medicines

* Check Near-Expiry Medicines

## Database

The project uses MySQL for storing user, pharmacist, and medicine records.
